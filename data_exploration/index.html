Data Exploration

The major objective of exploratory data analysis or data exploration is to gather initial insights by exploring the data gathered. This involves examining the contents of the dataset, handling missing values and outliers, ensuring consistency throughout the dataset, and performing initial tasks for Natural Language Processing (NLP), Time Series Analysis (TSA), data visualization, and feature analysis.

Overview

Preprocessing

In data pre-processing, we examine the contents of the dataset and identify key features relevant to our project, handle missing values and outliers, and ensure consistent formatting throughout the dataset.

Exploratory Analysis

We attempted to gather insights by performing basic NLP, TSA, and feature analysis tasks to make sense of the data we were able to gather..

Visualization

As part of our data exploration, we visualized the data through interactive plots that illustrate how the date a particular tweet was posted interacts with its type, poster, content, and other possible factors.
